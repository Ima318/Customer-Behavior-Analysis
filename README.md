📊 Data Analytics Portfolio Project


Overview
End-to-end data analytics project using Python, SQL, and Power BI — covering data loading, cleaning, exploration, querying, and dashboard reporting.

📁 Dataset
This dataset contains shopping behavior data for 3,900 customers, including demographic information, purchase details, product categories, review ratings, subscription status, discounts, payment methods, shipping preferences, and purchase frequency. It is designed for analyzing customer purchasing patterns, revenue trends, customer segmentation, and factors influencing buying behavior.



🛠️ Tools Used
ToolPurposePython (pandas)Data loading, EDA, cleaningSQL (PostgreSQL / MySQL / SQL Server)Querying and aggregationPower BIInteractive dashboardGammaPresentation / Report

🔢 Steps
Step 1 — Load Dataset

Imported raw CSV into Python using pandas
Checked shape, column names, and data types

Step 2 — Exploratory Data Analysis (EDA)

Identified distributions, outliers, and patterns
Used .describe(), .value_counts(), and visualizations

Step 3 — Data Cleaning

Handled missing values and duplicates
Standardized column formats and data types

Step 4 — SQL Queries

Loaded cleaned data into PostgreSQL / MySQL / SQL Server
Ran queries for aggregation, filtering, and joins
Extracted key business insights

Step 5 — Power BI Dashboard

Connected Power BI to the cleaned dataset
Built interactive visuals with slicers and filters
Highlighted top KPIs and trends

Step 6 — Report & Presentation

Created a structured report using Gamma
Sections: Problem → Data → Insights → Recommendations
